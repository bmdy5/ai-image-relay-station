# GPT-Image2 项目交付与待办功能清单 (MVP 路线图)

## 1. 项目当前状态 (Status)
- **技术栈**：FastAPI (Python 3.14) + React (Vite) + MySQL。
- **已打通链路**：用户注册 -> 登录 -> 鉴权守卫 -> 首页生图 -> 积分扣减 -> 管理员手动充值。
- **UI/UX**：亮色系高级感设计，基础框架已搭好。

## 3. 运行指南 (How to Run)
1. 启动命令：根目录下执行 `./start.sh`。
2. 环境变量：查看 `.env` 文件。
3. 数据库：MySQL 库名 `gpt_image_relay`。
