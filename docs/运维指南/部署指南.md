# Visionary 部署指南

## 1. 快速开始
项目采用 `start.sh` 进行一键式启动。

```bash
# 1. 配置免密登录 SSH (用于数据库隧道)
ssh-copy-id root@119.29.232.114

# 2. 修改 .env 配置文件
# 设置 DATABASE_URL, DASHSCOPE_API_KEY 等

# 3. 一键启动
./start.sh
```

## 2. 生产环境配置
*   **前端**: 运行 `npm run build` 后，将 `dist` 目录托管至 Nginx 或 Vercel。
*   **后端**: 建议使用 `pm2` 或 `systemd` 守护 `uvicorn` 进程。
*   **数据库**: 必须保持 SSH 隧道后台运行，或直接配置远程白名单直连。

## 3. 注意事项
*   端口占用：后端(8000), 前端开发(5173), 数据库隧道(3307)。
*   必须确保部署服务器具有足够的带宽用于图片的转存与下发。
