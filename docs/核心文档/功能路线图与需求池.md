# Visionary 功能路线图与需求池 (Product Roadmap)

本方案旨在将 Visionary 从简单的“中转站”升级为具备高度粘性与商业变现能力的“AI 艺术生产力平台”。

---

## 阶段 1: 专业化赋能 (MVP+) - 优先级：高
**目标**：提升生图可控性，减少积分浪费。

### 1.1 全参数控制面板 (Advanced Panel)
*   **需求**：在前端引入折叠面板，支持输入 `Seed` (种子)、`Negative Prompt` (负向词)。
*   **业务价值**：满足专业画师对构图一致性的要求，提升复购率。

### 1.2 画质修复与细节增强 (Enhancer)
*   **需求**：集成超分辨率接口，支持对已生成的图片进行 2x/4x 放大。
*   **业务逻辑**：此功能为二次消费项目，单次操作扣除 10 积分。

---

## 2. 社区与裂变 (Growth & Social) - 优先级：中
**目标**：通过用户内容带动平台流量增长。

### 2.1 灵感广场 (Prompter Gallery)
*   **需求**：用户可选择将生成的图片公开至广场。其他用户可点击“一键同款”复用其 Prompt。
*   **激进逻辑**：公开分享且获得 10 个赞的作品，奖励用户 5 积分，以此建立免费的优质内容库。

### 2.2 邀请分销体系 (Affiliate)
*   **需求**：二级分销。A 邀请 B，B 充值后 A 获得 10% 积分返利；B 邀请 C，A 获得 2% 积分返利。
*   **安全防护**：基于浏览器指纹与手机号双重校验，防止通过自动化脚本刷分。

---

## 3. 商业化进阶 (Monetization) - 优先级：高
**目标**：建立稳定的现金流。

### 3.1 会员分级订阅 (Membership)
*   **白银会员 (￥29/月)**：每日赠送 50 积分，解锁 HD 画质。
*   **黄金会员 (￥99/月)**：无限次数 Standard 生成，解锁 Master 画质，享独立渲染通道。

### 3.2 企业级 API 分销
*   **需求**：提供专用的 API Key 管理界面，允许第三方开发者直接接入我们的 API 接口进行二次开发。
