# UI 视觉进化 (UI Evolution) 功能设计文档

## 1. 功能概述
为 Visionary 平台增加一个专属的“UI 视觉进化”风格。该功能允许用户上传现有的 UI 草图或低保真截图，通过 AI 的视觉审计与审美重塑能力，一键生成符合现代大厂审美的高保真设计稿。

## 2. 核心特性
- **纯净一键式体验**：用户无需输入任何 Prompt，系统自动完成分析与美化。
- **布局一致性保证**：强制 AI 保留原稿的结构、导航和功能点位置。
- **视觉深度重构**：应用 Apple 风格与现代 Dribbble 流行趋势，优化光影、材质、圆角、间距和图标细节。

## 3. 技术实现

### 3.1 前端集成 (`frontend/src/pages/HomePage.jsx`)
- **风格 ID**: `ui_upgrade`
- **显示名称**: `UI 视觉进化`
- **描述**: `草图/截图一键转高保真大厂设计`
- **图标**: `🖼️`
- **占位符文本**: `💡 UI 进化模式：无需输入文字。请直接上传您的 UI 截图或草图，点击“开始创作”，系统将自动分析并重构。`
- **约束条件**: `requiresImage: true`, `pts: 'Master'` (归属旗舰版，30 积分)。

### 3.2 后端逻辑 (`backend/api/image.py`)
- **Prompt 模板**: 
  ```text
  你是一位资深 UI 设计师。请深度分析参考图中的 UI 布局和功能结构，你的任务是直接输出这张 UI 的视觉进化版。
  核心要求：
  1. 严禁改动布局：按钮在哪里，导航在哪里，必须保持原样。
  2. 视觉拉满：应用现代最高标准的 UI 审美，优化所有的间距（对齐）、圆角（统一）和阴影（多层呼吸感）。
  3. 组件重绘：将图中简陋的占位符图标替换为极具质感的现代矢量图标或 3D 图标。
  4. 材质升级：为界面加入细腻的材质，如轻微的磨砂玻璃效果、丝滑的渐变色和专业级的布光效果。
  请直接生成那张最完美的、高保真的、可以直接拿来做产品的 UI 设计稿。
  ```
- **包装引擎**: 针对 `ui_upgrade` ID，取消对用户输入的占位符替换逻辑，直接使用全量模板发送给 AI 接口。

## 4. 验收标准
- 用户不输入任何文字，仅上传图片，能够成功触发生成。
- 产出的图片在结构上与原图高度相似，但在色彩、细腻度和现代感上有肉眼可见的巨大提升。
- 计费逻辑正常，扣除 30 积分（Master 档位）。
