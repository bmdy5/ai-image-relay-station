# 新手引导系统设计规范 (New User Tutorial System)

> **状态**：🏗️ 计划中 (Draft)
> **日期**：2026-05-14
> **负责人**：Antigravity

## 1. 业务目标
通过交互式步进引导（A方案），帮助新注册用户快速上手生图流程，重点推介“提示词润色”功能，并引导用户进入“每日签到”留存闭环。

## 2. 核心交互流程 (5 步走)

| 步骤 | 目标组件 (ID) | 核心话术 | 交互重点 |
| :--- | :--- | :--- | :--- |
| 1. 灵感入口 | `guide-input-area` | 🎁 欢迎！输入你的灵感。如果想法太简单，点一下【提示词润色】即可变高级！别担心，点击撤销就能找回原稿。 | 强调润色和撤销功能 |
| 2. 审美定制 | `guide-style-list` | 为你的灵感挑选一个艺术风格，让画面更精准。 | 无 |
| 3. 见证奇迹 | `guide-generate-btn` | 一切就绪，点击点燃你的想象力。 | 渐变按钮高亮 |
| 4. 探索足迹 | `guide-history-tab` | 出图需要一小会儿，你可以在这里查看进度或回顾之前的作品。 | 缓解等待焦虑 |
| 5. 续航能力 | `guide-points-display` | 完成！记得去【每日签到】领取能量，持续你的创作之旅。 | 引导留存 |

## 3. 视觉规范 (Visual Specs)
*   **背景遮罩**：`rgba(0, 0, 0, 0.7)`，目标区域使用 `inset` 阴影或 `clip-path` 挖孔。
*   **引导气泡**：
    *   背景：`rgba(255, 255, 255, 0.85)` (毛玻璃效果)
    *   边框：`1px solid #C59C8F` (Visionary 主色)
    *   倒角：`16px`
    *   字体：主色调 `#333`, 重点词汇加粗使用 `#C59C8F`。
*   **动画**：气泡 `fade-in-up` 效果，高亮孔洞 `pulse` 呼吸动效。

## 4. 技术实现 (Technical Specs)

### 4.1 状态持久化
*   使用 `localStorage` 存储：`visionary_guide_v1: "finished"`。
*   **版本控制**：键名包含版本号（如 `v1`），未来重大功能更新可重置引导。

### 4.2 核心组件 `GuideController`
*   **职责**：管理当前步骤索引（index）、坐标计算、窗口 resize 重新定位。
*   **位置计算**：
    ```javascript
    const rect = targetElement.getBoundingClientRect();
    // 计算气泡位置，自动避开屏幕边缘
    ```

### 4.3 埋点记录
*   引导完成时，调用后端 API（可选）或前端日志，统计引导转化率。

## 5. 多端适配 (Responsive)
*   **PC 端**：气泡优先显示在目标元素的右侧或下方。
*   **移动端**：气泡优先显示在屏幕中心区域，箭头指向目标，避免遮挡键盘区域。

## 6. 验收标准
1. 用户首次登录必现引导。
2. 任意步骤点击“跳过”可立即关闭且不再出现。
3. 引导期间点击目标元素以外的区域应被屏蔽。
4. 移动端横屏/竖屏切换时，高亮位置能自动重绘对齐。
