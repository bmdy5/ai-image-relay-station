# 页面设计文档 (Page Design Spec)

## 1. 视觉风格与审美 (Aesthetics)
*   **风格定位**：高端暗黑风 (Dark Mode)，极简主义，强调生成出的图片质感。
*   **核心配色**：深灰 (#121212) 作为主背景，亮紫/青色作为交互点缀，图片展示区使用毛玻璃效果 (Glassmorphism)。
*   **字体**：优先使用 Inter 或系统默认的无衬线字体，字间距略微拉大以体现高级感。

## 2. 核心页面功能 (Core Features)

### 2.1 登录与注册 (Auth)
*   **注册限额**：系统硬性限制前 100 名用户，超出后注册按钮置灰。
*   **表单字段**：账号、密码、邀请码（可选）。
*   **引导语**：首批内测名额稀缺，仅限 100 人。

### 2.2 首页 (Home)
*   **营销区 (Why Choose Us)**：
    *   **官方门槛**：OpenAI 官方 API 需要 Visa 卡和海外环境，门槛极高。
    *   **订阅成本**：ChatGPT Plus 会员每月 $20 (约 145 元)，且有额度限制。
    *   **我们的优势**：免 Visa、国内直连、按需计费（10 元起充）、支持高精度 Thinking 模式。
*   **提示词输入框**：支持多行输入。
*   **规格选择器**：
    *   **低质量**：默认选项，极速出图。
    *   **中质量**：提示高精细度。
    *   **高质量**：最高画质。
    *   **模式切换**：Instant（当前启用） / Thinking（第一版置灰提示暂不支持）。
*   **生成按钮**：点击后显示动效（如：正在解析提示词... -> 正在绘图中...）。
*   **结果展示区**：支持图片点击放大、一键下载、复制图片链接。
*   **用户状态栏**：
    *   显示“我的识别码”（Fingerprint）。
    *   显示“当前剩余积分”。
    *   充值按钮（点击弹出充值说明）。

### 2.2 充值与私域转化弹窗 (Payment/Conversion)
*   **触发条件**：点击充值按钮，或积分不足以进行当前生成的规格。
*   **内容**：
    *   展示客服微信二维码。
    *   展示积分价格表（10元=100积分等）。
    *   文字引导：“加客服微信发送识别码并付款，秒级人工到账”。

### 2.3 管理员后台 (`/admin`)
*   **权限拦截 (Route Guard)**：
    *   **前端拦截**：禁止直接通过 URL 访问。若用户未登录或 `is_admin` 为 false，访问此路由立即重定向至首页。
    *   **入口隐藏**：普通用户看不到进入后台的按钮，仅管理员登录后在个人中心可见。
*   **二次验证**：进入后台操作充值等敏感功能时，需再次输入“管理员安全码”。
*   **操作面板**：
    *   一键充值按钮（+50, +100, +500）。
    *   日志监控：实时滚动显示全站生图动态。

## 3. 安全风险控制 (Security & Risk)
*   **接口保护**：
    *   所有 API 请求必须携带前端生成的 Fingerprint。
    *   后端对单个 Fingerprint 设置请求频率限制（QPS），防止爬虫。
*   **内容审计**：
    *   接入简单的敏感词库，命中敏感词直接提示“该提示词违反社区规定”。
*   **后端熔断**：
    *   设置每日 API 消费上限（例如 $50），保护账号。

## Requirements
### Requirement: 消费总额熔断
系统必须支持设置单日全站 API 消费上限。

#### Scenario: 达到单日限额停止服务
- **WHEN** 今日全站生图消耗的 API 总金额达到预设的熔断线（如 $50）
- **THEN** 系统必须暂时停止所有生图请求，并向所有用户提示“系统维护中”。

### Requirement: 提示词敏感词拦截
系统必须在调用 OpenAI 之前拦截包含违规词汇的提示词。

#### Scenario: 命中敏感词拦截
- **WHEN** 用户输入的提示词包含系统预设的违规词汇
- **THEN** 系统必须拦截该请求，不调用外部 API，并提示“词汇违规”。
