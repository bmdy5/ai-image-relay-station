# GPT-Image2 未来升级与演进计划 (Phase 2+)

本文档用于记录项目在 MVP 版本之后的中长期升级构思，确保核心功能的演进方向一致。

## 1. 创作体验增强
- [ ] **Prompt 智能助手**：接入 LLM（如 GPT-4o-mini）帮助用户优化、润色生图提示词。
- [ ] **一键复用 (Reuse)**：历史记录支持“一键填入旧 Prompt”，方便基于旧创意进行微调。
- [ ] **历史搜索**：支持按 Prompt 关键词、日期、画质规格搜索个人创作记录。
- [ ] **高级编辑 (Advanced Editing)**：
    - [ ] 局部重绘 (In-painting)：通过涂抹修改图片局部。
    - [ ] 扩图 (Out-painting)：向四周延伸画作内容。
    - [ ] 变体生成 (Variation)：生成 4 张风格相近但细节不同的候选图。

## 2. 支付与商业化演进
- [ ] **个人收款码回调自动化**：引入第三方监控工具，实现个人转账后的毫秒级自动到账，彻底取代人工审核。
- [ ] **会员订阅制 (SaaS)**：推出周费/月费计划，享受每日免费额度或生图折扣。
- [ ] **分销/邀请奖励**：用户邀请新用户注册并充值后，可获得一定比例的积分返现。

## 3. 安全与风控升级
- [ ] **异常行为分析**：识别短时间内大量消耗积分的“机器人”账户，自动临时封禁。
- [ ] **邮箱/手机号绑定**：增加安全的账户找回机制，提高账户权重。
- [ ] **多设备管理**：支持查看当前在线设备，并可远程强制下线。

## 4. 社区与分享
- [ ] **广场 (Gallery)**：允许用户将满意的画作一键公开发布到广场，供其他用户点赞、复制 Prompt。
- [ ] **个人主页展示**：用户可配置自己的公开主页，展示其艺术风格。

---

## 5. 技术债务与架构优化
- [ ] **Redis 缓存**：引入 Redis 存储高频访问的用户信息和 API 限流计数器。
- [ ] **图片存储 CDN**：将生成的图片自动上传至云存储（OSS/S3），加速访问并降低主服务器带宽压力。
- [ ] **单元测试覆盖**：对核心生图逻辑和积分扣减逻辑实现 90% 以上的测试覆盖。
