## Why

用户在输入提示词时往往由于缺乏专业词汇或细节描述能力，导致生成的图片质量不稳定。通过引入基于大语言模型（如通义千问）的智能助手，可以降低用户的创作门槛，将简短的意图自动扩充为高质量、细节丰富的专业提示词，并辅以极具仪式感的视觉动画，提升平台的高级感和用户留存。

## What Changes

- **后端集成智能润色引擎**：新增独立模块调用 LLM（优先通义千问），支持语种自适应。
- **风格感知优化**：优化器根据用户当前选择的风格模版进行针对性扩充。
- **前端交互增强 (WOW效果)**：
  - **输入框呼吸灯动画**：优化过程中，输入框呈现流动的彩色渐变边框特效。
  - **非流式快速替换**：点击优化后，整体替换文字，配合淡入动画。
  - **撤销/恢复功能**：优化后提供“撤销”按钮，一键找回原始输入。
  - **覆盖保护**：切换风格时检测到已有内容则弹出确认提示。
- **安全与限流**：单用户 5 次/分钟的速率限制。

## Capabilities

### New Capabilities
- `prompt-enhancement-engine`: 提供上下文感知的提示词润色能力。
- `api-security-limits`: API 频率控制。
- `ux-animation-system`: 专门用于 AI 交互的视觉反馈动画系统。

### Modified Capabilities
- `image-generation`: 扩充交互闭环。

## Impact

- **API**: `POST /image/enhance-prompt`
- **Frontend**: 新增全局 CSS 变量（用于呼吸灯色调）及组件状态管理。
