# 手机端专业版 (Mobile Pro V2) 深度升级方案

## 1. 背景与目标 (Context & Goals)
将 AI Image Relay Station 的手机端 UI 从“普通网页感”升级为“顶级 AI 创作助手”。通过引入对话式结果流、智能功能坞以及 PC 端大师模式的特权反馈，提升移动端的操作效率与视觉档位。

## 2. 核心设计规范 (Design System)
- **字体**：全局启用 `Outfit` 字体。
- **圆角**：标准化大圆角 `24px (lg)` 与功能组件圆角 `16px (md)`。
- **配色**：沿用亮橙色 (`var(--primary)`) 作为交互色，电光紫 (`var(--master)`) 作为大师版专属色。
- **布局**：
    - **Header**：固定顶部，毛玻璃背景。
    - **Stack**：垂直瀑布流式结果堆叠。
    - **Dock**：底部悬浮功能坞 + 胶囊输入框。

## 3. 页面详细设计 (Page-by-Page Design)

### 3.1 首页 (Home - The Workshop)
- **对话流交互**：
    - 结果以垂直卡片序列排列，最新结果置底。
    - 发送指令后，页面平滑滚动到底部。
- **大师版视觉沉淀**：
    - 在生成中的卡片内展示粒子流动画与“思维矩阵”关键词。
    - 生成后提供 `✦` 创作笔记入口，通过底部抽屉展示 AI 深度优化详情。
- **智能功能坞 (Function Dock)**：
    - **档位切换**：弹出半屏抽屉，展示标准/高清/大师的对比及积分消耗。
    - **风格实验室**：弹出 3 列网格抽屉，支持带图预览的风格选择。
    - **高级设置**：根据档位锁定，提供宽高比切换。

### 3.2 历史画廊 (History - The Gallery)
- **双列瀑布流**：改用两列式网格布局，提高浏览效率。
- **状态标记**：卡片上浮动显示 `Standard/HD/Master` 标签。
- **快捷复用**：点击卡片进入详情，支持一键复用提示词并跳转回首页。

### 3.3 会员与权益 (Pricing - The Vault)
- **横向卡片系统**：积分套餐采用横向滑动卡片，强化购买时的沉浸感。
- **权益透明化**：通过对比列表明确“大师版”在移动端的独特价值（专属笔记、高并发、4K 增强）。

### 3.4 个人中心 (Profile - The Dashboard)
- **极简看板**：以粗体数字和进度条展示积分余额与创作统计。
- **分组列表**：按照 Apple Settings 风格重组功能菜单（账户、安全、反馈）。

## 4. 技术实施细节 (Implementation)
- **组件重构**：主要修改 `MobileHomePage.jsx`, `HistoryPage.jsx`, `PricingPage.jsx`, `ProfilePage.jsx`。
- **布局适配**：更新 `MobileLayout.jsx` 确保全局底部导航与新版“功能坞”不冲突。
- **动画引擎**：使用 CSS Keyframes 处理抽屉入场、粒子流及生图脉冲效果。

## 5. 验收标准 (Success Criteria)
- 手机端首页可实现“生成 -> 堆叠 -> 继续生成”的闭环。
- 大师版生图过程具有明显的动态视觉反馈。
- 全站四张主要页面的视觉语言高度统一。
