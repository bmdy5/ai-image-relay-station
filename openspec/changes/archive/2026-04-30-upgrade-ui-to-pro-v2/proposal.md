## Why

当前 UI 布局被用户认为不够美观且专业感不足。为了提升用户体验并强化“标准、高清、大师”三个档位的差异化价值（如风格模板、图生图、深度思考等），需要进行一次深度的 UI 重构和功能升级。

## What Changes

- **界面架构重组**：从单一垂直布局改为侧边栏控制 + 右侧沉浸式预览的布局。
- **阶梯化功能门控**：
    - **标准版**：仅支持默认与写实双风格，锁定高级工具。
    - **高清版**：解锁“风格实验室”全选弹窗，解锁图生图上传位。
    - **大师版**：全功能开放，新增“AI 深度思考”开关。
- **视觉反馈系统**：新增大师版专用的“思维矩阵”粒子特效和生成后的“创作笔记”悬浮卡片。
- **UI 质感提升**：引入 Outfit 字体、20px-40px 大圆角、星芒图标以及高级灰调视觉风格。

## Capabilities

### New Capabilities
- `style-lab-modal`: 一个沉浸式的毛玻璃弹窗，用于挑选各种艺术风格模板。
- `master-mode-insights`: 大师模式专属的视觉反馈层，包含关键词流转动画和创作笔记分析。

### Modified Capabilities
- `visual-design-system`: 更新全局设计规范（字体、圆角、调色板）。
- `image-quality-tiering`: 更新不同档位下的前端展示逻辑和功能可用性。
- `page-design.md`: 调整主页面的核心布局结构。

## Impact

主要影响前端 UI 层（HTML/CSS/JS），包括主页面结构、模式切换的状态管理逻辑以及新增的交互组件。不涉及后端生成接口的协议变更，但会影响前端如何调用和展示这些功能。
