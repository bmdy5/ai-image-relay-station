## ADDED Requirements

### Requirement: 风格预填联动
当用户在风格实验室选择一个新风格时，前端必须自动将对应的 `UI 预填文本` 填充至输入框。

#### Scenario: 自动填充
- **WHEN** 用户点击 "旅游海报" 风格且输入框为空
- **THEN** 输入框内容自动变为 "城市名称：【在此输入城市，如：重庆】"

### Requirement: 覆盖内容二次确认
若用户已输入自定义内容并尝试切换风格，系统必须弹窗询问是否覆盖。

#### Scenario: 覆盖确认
- **WHEN** 输入框已有 "一只猫" 且用户选择 "绘梨衣" 风格
- **THEN** 弹出对话框确认 "是否应用新风格的提示词模版？这会覆盖您当前的内容。"

### Requirement: 全等级比例选择器
系统必须在“参数微调”区域提供 1:1, 9:16, 16:9 三个比例选择按钮，且对所有等级用户开放。

#### Scenario: 比例切换
- **WHEN** 用户选择 "16:9" 按钮
- **THEN** 发送给后端的 `aspect_ratio` 参数变为 `16:9`
