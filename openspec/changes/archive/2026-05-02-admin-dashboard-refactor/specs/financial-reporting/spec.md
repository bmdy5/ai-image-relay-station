## ADDED Requirements

### Requirement: 财务数据汇总
系统必须能够按时间段（日、周、月）汇总全站成功的生图积分消耗。

#### Scenario: 管理员查看本月积分消耗
- **WHEN** 管理员在仪表盘选择“本月”时间范围
- **THEN** 系统返回所有 `status='success'` 的任务所消耗的积分总和

### Requirement: 利润计算器
系统必须允许管理员手动输入外部成本（API 账单），并结合充值收入计算净利润。

#### Scenario: 计算月度净利润
- **WHEN** 管理员输入本月 API 账单金额为 500 元
- **THEN** 系统展示：(总充值金额) - 500 = 净利润，并以表格形式呈现。
