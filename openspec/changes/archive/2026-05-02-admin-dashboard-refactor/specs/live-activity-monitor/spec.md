## ADDED Requirements

### Requirement: 全站实时生图流
管理员仪表盘必须实时展示全站用户的生图动态，包含提示词、用户信息及生成结果预览。

#### Scenario: 监控最新动态
- **WHEN** 有新用户生成图片成功
- **THEN** 管理员仪表盘的“实时流”列表自动更新，显示最新生成的图片和 Prompt。

### Requirement: 违规内容抹除
管理员必须能够直接从监控流中一键删除违规记录及其关联的物理文件。

#### Scenario: 抹除违规图片
- **WHEN** 管理员点击某条记录的“彻底抹除”按钮
- **THEN** 系统从数据库删除该记录，并调用存储 API 删除云端原文件。
